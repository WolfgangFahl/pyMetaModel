## Introduction
pyMetaModel is a python library for a Knowledge Graph MetaModel

It started with the ProfiWiki [SemanticMediaWiki MetaModel](https://wiki.bitplan.com/index.php/MetaModel#tab=topics) in
2015 consisting of

- Context
- Topic
- TopicLink
- Property
- Action
- SMWType
